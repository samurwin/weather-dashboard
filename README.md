# weather-dashboard

## Purpose
A deployed website that allows you to search for the weather in a particular city, showing you the current weather and 5 day forecast.

## Built With 
* HTML
* CSS
* Javascript 
* jquery
* bootstrap
* day.js
* One Call Weather API
